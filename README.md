# 🎓 Student Expense Tracker

A lightweight, desktop-based GUI application built in Python using **Tkinter** and **SQLite3**. Designed specifically for students to manage their budgets, categorize dynamic expenses, and view real-time spending insights.

---

## 🚀 Key Features
* **Interactive Dashboard:** Core navigation hub for seamless window switching.
* **Smart Data Input:** Form validation with drop-down categories and an integrated calendar module.
* **Live Ledger Display:** Dynamic data population via a `Treeview` data matrix pulled straight from SQLite.
* **On-Demand Metrics:** (In Development) Aggregated summary metrics and spending breakdowns.

---

## 🛠️ Built With
* **Language:** Python 3.x
* **Database:** SQLite3
* **GUI Framework:** Tkinter / ttk
* **Additional Libraries:** `tkcalendar`
